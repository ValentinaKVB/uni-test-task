Представленный код 4 раза выводит "Bad: undefined"

1-й вариант модификации кода:
Изменить в цикле var на let, 
чтобы при каждой итерации инициализировалась 
новая переменная с блочной областью видимости, 
а не переназначалось значение одной и той же
глобальной переменной.

2-й вариант модификации кода:
Обернуть setTimeout() дополнительной функцией
и ввести в ней новую локальную переменную, которой присвоить
значение переменной i. Затем вызвать эту функцию внутри цикла.

for (var i = 0; i < arr.length; i++) {
  function wrapTimeout() {
    var j = i;
    setTimeout(function () {
      console.log(arr[j] > 13 ? `Good: ${arr[j]}` : `Bad: ${arr[j]}`);
    }, 3000);
  }
  wrapTimeout();
}
